This project uses five buttons to control four LEDs using fast hardware interrupts. The left and right buttons switch between three different blinking patterns, while the top and bottom buttons change how fast or slow the lights flash. The center button acts as a main power switch to turn the blinking completely on or off. We use interrupts to respond instantly without slowing down the system.
